Project Overview

The Virtual Event Ticketing System is an ASP.NET Core MVC web application that allows users to browse virtual events, purchase tickets as guests, and manage event listings. The system features a modern, responsive design and integrates with a PostgreSQL database for data persistence.

Core Features Implemented

Event Management

  • Complete CRUD operations (Create, Read, Update, Delete)
  • Event details: title, category, date/time, ticket price, availability
  • Color-coded cards for available, low-stock (< 5 tickets), and sold-out events
  • Category organization (Webinars, Concerts, Workshops, Conferences)

Search and Filtering

  • Case-insensitive text search by event title or date
  • Filter by specific categories
  • Date range filtering (start and end dates)
  • Availability filter (available vs. sold-out)
  • Sort by title, date, or price (ascending/descending)

Guest Ticket Purchasing

  • Browse and select from available events
  • Choose number of tickets to purchase
  • Enter guest information (name and email - no registration required)
  • Purchase confirmation with confirmation number and details
  • Automatic ticket inventory management

Dashboard

  • Display total events and categories
  • Low ticket alerts for events with fewer than 5 tickets
  • Track sold-out events
  • Show next 5 upcoming events ordered by date

Responsive Design

  • Bootstrap 5 for modern, mobile-friendly interface
  • Responsive navigation with collapsible menu
  • Card-based layout for clean presentation
  • Color-coded UI with visual feedback

Database Integration

  • PostgreSQL database with Entity Framework Core
  • Four core tables: Categories, Events, Purchases, PurchaseItems
  • EF Core migrations for schema versioning
  • Model-level validation with data annotations
  • Proper foreign key relationships and referential integrity

Technology Stack

ASP.NET Core 8.0 MVC PostgreSQL Entity Framework Core Bootstrap 5 HTML5 CSS3 JavaScript Git & GitHub

Key Highlights

  • Complete CRUD operations for event management

  • Advanced search and filtering capabilities
